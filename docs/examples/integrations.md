# Framework Integrations

Learn how to integrate Atomix with Next.js (both App Router and Pages Router), handle server-side rendering, and optimize performance.

- [Next.js Integration Guide](../NEXTJS_INTEGRATION.md)

## Astro

Learn how to use Atomix React components within an Astro project, manage client-side hydration for interactive components, and customize the global styles.
